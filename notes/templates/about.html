All About Notes Sharing

“PathSala” is the project used for sharing notes using the website. Notes that the user wants for a particular subject can be found in the website. This project is written in Python Django and sqlite. Firstly, the user need to sign up and then, they can log in to the system.



“PathSala” is a open source you can Download and edit as per you need.
